Module 02 · Agent patterns

Coming in the next session

Workflow vs agent vs chain. Subagents. Orchestrator-worker, evaluator-optimizer, parallel patterns. Where the autonomy budget actually sits.

This module is locked. Mihailo authors one module per Claude Code session. When this one ships, this page will be replaced with the full content: prose, mermaid diagrams, interactive demos, sandboxes with hidden tests, glossary, check questions, and a completion banner.

What this module will cover

The taxonomy of agent shapes (one-shot prompt, chain, workflow, agent, multi-agent), how to choose between them, when to spawn subagents, and the four canonical patterns from Anthropic's research: orchestrator-worker, evaluator-optimizer, parallelization, and routing. Live diagrams of each pattern, sandbox where you implement a basic orchestrator, and a tour of how Claude Code uses subagents under the hood.
